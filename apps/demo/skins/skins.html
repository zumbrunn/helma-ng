<% extends 'base.html' %> <% subskin 'content' %> This page showcases some of the skinning featuers in Helma NG, such as macros, subskins, and filters.

This is the action code that renders the skin:

function skins_action() {
    var context = {
        name: 'Luisa',
        names: ['Benni', 'Emma', 'Luca', 'Selma']
    };
    res.write(render('skins/skins.html', context));
}

And the following subskin in skins.html:

<% subskin 'hello' %>
Hello <% name %>!

Your basic skin rendering macro then looks like this:

<% render 'hello' %>

And this is the output this produces:

<% render 'hello' %>

The render macro also lets you loop through arrays and other iterables:

<% render 'hello' on=<% names %> as='name' %>

This is what you should get:

<% render 'hello' on=<%names%> as='name' %>

You can also bind values to the macro context in the tag: 
<% render 'hello' bind={ name: "Julia" } %>

This gets you

<% render 'hello' bind={ name: 'Julia' } %>

Or you can define the objects to loop through using object and array literals in the macro tag:

<% render 'hello' on=['Gerlinde', 'Thomas'] as='name' %>

Produces:

<% render 'hello' on=['Gerlinde', 'Thomas'] as='name' %>

Finally, filters let you do all kinds of post-processing on the output of a macro.

<% render 'hello' | uppercase %>

Renders:

<% render 'hello' | uppercase %>
